📖 Hướng dẫn dùng

  1. TikTok Ads: Ads Manager → chọn Ad → Tab Ad Details → Trường URL (hoặc Website URL) → Paste URL đã tạo → Save → Publish. TikTok sẽ REVIEW ads lại (thường 1-2h).

  3. Google Ads: Campaign → Ad → Edit → Final URL → Paste.
